# Function---Methods

A collection of Java programs focused on mastering functions, methods, and modular code architecture. This repository documents the implementation of reusable logic and structured input handling to strengthen programming fundamentals.

## 🚀 Goal
To master Java methods, parameter passing, and modular logic, moving from basic syntax to functional programming techniques.

## 📁 Programs Implemented

- ✅ **Calculate Sum**: Function-based logic to compute the sum of two integers.
- ✅ **Calculate Product**: Implementation of multiplication using methods.
- ✅ **Calculate Factorial**: Iterative approach to calculate the factorial of a given number.
- ✅ **Print Name**: Demonstrating basic function definition and invocation to process user input.

## 🛠 Concepts Covered

- **Method Definition**: Creating and calling functions for modularity.
- **Parameters & Arguments**: Passing data into functions.
- **User Input**: Utilizing the `Scanner` class for interactive programs.
- **Return Values**: Handling data output from functions to the `main` method.
